## Para que ele serve

O EduSaaS foi projetado para resolver **todas as necessidades operacionais** de uma escola de cursos profissionalizantes. Veja como cada problema é endereçado:

| Necessidade | Como o EduSaaS resolve |
|---|---|
| **Captar novos alunos** | Módulos de Telepesquisa e Teleprojeto para prospecção e agendamento |
| **Organizar visitas** | Recepção digital com controle de status em tempo real |
| **Realizar matrículas** | Comercial com fluxo guiado de atendimento e fechamento |
| **Gerar contratos** | Administrativo com geração automática de contrato em PDF |
| **Controlar pagamentos** | Financeiro com entradas, saídas, conciliação bancária e boletos |
| **Acompanhar alunos** | Pedagógico com chamadas, pendências, disparos de WhatsApp e eventos |
| **Encaminhar para emprego** | RH com cadastro de empresas, vagas e carta de encaminhamento |
| **Gerenciar múltiplas unidades** | Multifiliais com dashboards comparativos |
| **Aulas digitais** | Área do Aluno com conteúdo interativo (parceria 7way Idiomas) |
| **Monitorar produtividade** | Dashboards, metas e métricas individuais em todos os módulos |

<Callout type="tip" title="Tudo conectado">
Os dados registrados em uma etapa não precisam ser digitados novamente na etapa seguinte. Tudo é herdado automaticamente entre os módulos.
</Callout>
