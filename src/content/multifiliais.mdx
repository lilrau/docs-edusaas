## Multifiliais

> **Quem usa:** Administrador (Admin)
> **Objetivo:** Visão consolidada de todas as unidades/filiais/franquias. Comparar desempenhos.

### Comercial

Acesse **Multifiliais** → **Comercial** para ver o desempenho de matrículas de todas as filiais lado a lado.

### Pedagógico

Acesse **Multifiliais** → **Pedagógico** para comparar frequência, retenção e indicadores pedagógicos entre filiais.

### Financeiro

Acesse **Multifiliais** → **Financeiro** para comparar entradas, saídas e saldo entre todas as filiais. Nesta tela também é possível:

- Ver detalhes financeiros de cada filial.
- Configurar **metas financeiras** por filial.
- Editar e remover configurações de metas.

### Líder Tele

Acesse **Multifiliais** → **Líder Tele** para comparar o desempenho de captação e agendamentos entre as unidades.

<Callout type="tip" title="Pro Tip: Alternância entre filiais">
O admin pode **alternar entre filiais** livremente para visualizar e gerenciar cada unidade de forma independente. Use isso para identificar rapidamente quais unidades precisam de atenção.
</Callout>
